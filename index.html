About Smart Library

Overview

Smart Library is a modern prototype that helps readers discover books fast and helps admins manage collections with clarity. It brings search, availability, and day-to-day library workflows into one clean experience. Users can log in, browse, and act quickly without losing context. Admins can keep the catalog accurate with simple add and update flows. The interface focuses on speed, simple navigation, and visual discovery. The project blends a classic C backend with a polished web layer for a practical end-to-end demo.

- Sahil (B.Tech, Cybersecurity)
